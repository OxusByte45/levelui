# Migration Notice

This repository has been completely modernized from the original [heapwolf/levelui](https://github.com/heapwolf/levelui).

## What Changed

- ✅ **Modernized**: Electron 0.28.1 → 39.2.3
- ✅ **Security**: Added context isolation and secure IPC
- ✅ **Build System**: Migrated to electron-vite
- ✅ **Code Editor**: Upgraded to CodeMirror 6
- ✅ **Structure**: Reorganized into modern Electron architecture

## Quick Start

```bash
npm install
npm run dev    # Development with hot reload
npm start      # Build and run
```
